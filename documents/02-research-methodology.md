# 02 — 研究方法論

## 基本模型：借貸投資模擬

### 模擬流程

```
Day 0: 貸款撥款 → 全額買入標的 ETF
       ↓
Month 1~N: 每月循環
  ├─ 檢查是否有股利入帳 → 依策略處理（再投入/用於還款）
  ├─ 檢查是否有公司行動（分割/合併）
  ├─ 計算本月應還款金額（本金 + 利息）
  ├─ 若現金不足 → 賣出最少需要的股數
  ├─ 支付月還款
  └─ 記錄：持股數、市值、淨資產、現金餘額
       ↓
Month N (最後一期): 還清貸款
  ├─ 計算剩餘股票市值 + 剩餘現金
  ├─ 扣除所有成本（利息、交易費、稅）
  └─ 得出最終淨損益
```

### 關鍵假設

| 假設項目 | 設定值 | 說明 |
|----------|--------|------|
| 買入時機 | 撥款日收盤價 | 假設當日全額買入 |
| 賣出時機 | 每月 13 日（或最近交易日） | 在還款日前賣出 |
| 成交價格 | 收盤價 | 簡化假設，實務可能有價差 |
| 買入手續費 | 0.1425% | 法定上限，實務可能有折扣 |
| 賣出手續費 | 0.1425% | 同上 |
| 證交稅 | 0.1% | ETF 優惠稅率 |
| 股利處理 | 再投入買股 | 除非接近還款日 |
| 交易單位 | 依時期而定 | 2005-03-01 後可零股交易；2005-03-01 前僅能整張（1,000 股）交易，詳見下方說明 |

### 交易單位的時期差異

回測模擬需依據歷史上零股交易的開放時間，採用不同的賣出邏輯：

| 時期 | 交易制度 | 回測邏輯 |
|------|---------|---------|
| **2005-03-01 之前** | 僅整張交易（1,000 股） | 每月賣出 `Math.ceil(需求金額 / 每張淨收入)` 張，多餘現金留存至下月一併計算（不買回，因零頭不足一張） |
| **2005-03-01 之後** | 盤後零股交易開放 | 每月賣出 `Math.ceil(需求金額 / 每股淨收入)` 股，幾乎無閒置現金 |

> **說明**：台灣盤後零股交易自 2005-03-01 起開放，2020-10-26 再開放盤中零股交易。由於本模型使用收盤價計算且需考量 T+2 交割（提前 2~3 天賣出），盤中與盤後零股在模型中無差異。
>
> **影響範圍**：僅 0050 在 2003-07 至 2005-02 期間的回測受此限制，需以整張交易模擬。此時 0050 股價約 37~58 元，一張約 37,000~58,000 元，月還款約 20,000 元，每月賣出一張會產生約 17,000~38,000 元的閒置現金，對績效有一定拖累。

## 分析維度一：歷史期間回測

### 滾動回測法
- 以每個月的第一個交易日作為假想的貸款撥款日
- 模擬完整的 84 期（7年）還款週期
- 記錄每個進場時間點的最終結果

### 分析範例（以 0050 為例）
```
進場 2010-01: 經歷歐債危機 → 結果 A
進場 2015-01: 經歷中美貿易戰 → 結果 B
進場 2018-01: 經歷 COVID-19 → 結果 C
進場 2020-03: 疫情低點進場 → 結果 D
...
```

### 產出指標
- 勝率：多少比例的進場時間點最終獲利
- 平均報酬率與中位數報酬率
- 最佳/最差進場時間點及其結果
- 還款期間最大淨資產回撤

## 分析維度二：情境模擬

### 五種市場情境

#### 1. 牛市情境
- 年化報酬率：15~20%
- 年波動率：15%
- 代表期間：2020-03 至 2021-12

#### 2. 正常成長情境
- 年化報酬率：8~12%
- 年波動率：20%
- 代表期間：長期歷史平均

#### 3. 盤整情境
- 年化報酬率：0~5%
- 年波動率：15%
- 代表期間：2015-04 至 2016-12

#### 4. 熊市情境
- 年化報酬率：-10~-20%
- 年波動率：30%
- 代表期間：2008 金融海嘯

#### 5. 黑天鵝情境
- 短期暴跌 30~50%，之後逐步回升
- 代表事件：2020-02 至 2020-03 COVID-19

### 蒙地卡羅模擬（進階）
- 基於歷史月報酬率的分布，隨機生成 10,000 條價格路徑
- 統計最終淨損益的分布
- 計算特定信心水準下的最差結果（如 95% VaR）

## 分析維度三：還款壓力測試

### 現金流斷裂風險
- 定義：某月賣出所有剩餘持股仍無法支付月還款
- 衡量指標：在各情境下，現金流斷裂的機率
- 觸發條件：股價跌幅 × 已賣出比例 → 剩餘股數不足

### 心理壓力指標
- 帳面虧損超過貸款金額 50% 的月份數
- 連續帳面虧損的最長月份數
- 需要額外掏錢還款（自掏腰包）的月份數

### 安全邊際計算
- 在各情境下，保留多少現金儲備可避免斷裂
- 每月額外投入多少金額可顯著降低風險
