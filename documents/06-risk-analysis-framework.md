# 06 — 風險分析框架

## 借貸投資的特殊風險結構

借貸投資與一般投資最大的差異在於：**有固定的現金流出（月還款）且無法暫停**。

這意味著：
- 即使長期看好，短期下跌可能迫使你在低點賣股
- 賣在低點 → 持股減少 → 未來反彈時的收益也減少 → 「跌時多賣、漲時少持」的負面循環
- 這是借貸投資最核心的結構性風險

## 風險分類

### 1. 市場風險（系統性風險）

**定義**：整體市場下跌導致持股市值縮水

**衝擊機制**：
```
市場下跌 → 股價下跌 → 每月需賣出更多股數才能湊足月還款
→ 持股加速減少 → 即使市場回升，持股已不足以回本
```

**量化指標**：
- 最大回撤 (MDD)：持有期間淨資產從高點到低點的最大跌幅
- 波動率：月報酬率的標準差
- Beta：相對於大盤的波動倍數

**各標的風險等級**：
| 標的 | 市場風險等級 | 說明 |
|------|------------|------|
| 0050 | 中 | 與大盤高度連動，波動率適中 |
| 0056 | 中低 | 高股息股通常波動略低 |
| 00631L | 極高 | 2倍槓桿 + 波動率拖累 |
| 00679B | 中 | 受美國利率政策影響，與股市低相關 |

### 2. 流動性風險

**定義**：需要賣股時無法以合理價格成交

**衝擊場景**：
- 極端恐慌時市場流動性枯竭
- 標的成交量過低，大量賣出會衝擊價格
- ETF 折溢價在極端情境下可能擴大

**量化方式**：
- 平均日成交量 vs 每月需賣出金額
- 估算賣出對市場的衝擊成本

### 3. 利率風險

**定義**：貸款利率上升導致月還款增加

**衝擊計算範例**（150萬/84期）：
| 利率 | 月付金 | 總利息 | 比基準增加 |
|------|--------|--------|-----------|
| 3.00% | 19,804 | 163,514 | -72,419 |
| 4.23% | 21,585 | 235,933 | 基準 |
| 5.00% | 22,449 | 285,731 | +49,798 |
| 6.00% | 23,545 | 347,816 | +111,883 |

### 4. 現金流斷裂風險

**定義**：某月無法湊足月還款金額

**觸發條件**：
```
剩餘股數 × 當日股價 × (1 - 交易成本) + 現金餘額 < 月還款金額
```

**後果**：
- 需自掏腰包補足差額
- 若無法補足，可能逾期 → 信用受損 → 被催收

**預防措施**：
- 計算每個情境下的「安全墊」需求
- 建議保留 3~6 個月的月還款金額作為緊急預備金

### 5. 時序風險（Sequence of Returns Risk）

**定義**：同樣的平均報酬率，不同的報酬時序會導致截然不同的結果

**關鍵洞察**：
- 如果前期就遇到大跌，大量股票在低點被賣掉還款
- 後期即使大漲，因為持股已大幅縮水，漲幅的絕對收益有限
- 反之，前期上漲 → 少賣股 → 保留更多持股 → 後期即使下跌也有較厚的緩衝

**模擬方式**：
- Bootstrap 模擬自然捕捉此風險：5,000 次抽樣會產生各種不同的漲跌順序排列
- 悲觀情境（P25）往往是「前期遇到大跌」的路徑，樂觀情境（P75）則反之
- 從 P25 與 P75 的差距可量化「運氣」（漲跌順序）在借貸投資中的權重

### 6. 00631L 特有風險：波動率拖累

**定義**：槓桿 ETF 追蹤的是「每日報酬的 N 倍」，長期持有會因波動率而產生淨值損耗

**範例**：
```
0050 連續兩天：+10%, -10% → 淨值變化 = 0.99 (-1%)
00631L 連續兩天：+20%, -20% → 淨值變化 = 0.96 (-4%)
```

**在借貸投資情境下的影響**：
- 盤整市場中，00631L 的淨值會持續損耗
- 借貸成本（利息）是固定的，但資產在縮水
- 只有在強勢單邊上漲時才有優勢

## 風險量化工具

### Value at Risk (VaR)
- 在 95% 信心水準下，最大可能虧損金額
- 以 Bootstrap Monte Carlo 模擬計算（5,000 次，從歷史月報酬率有放回抽樣）

### Conditional VaR (CVaR / Expected Shortfall)
- VaR 之外的平均虧損（尾部風險）
- 更能反映極端情況的損失
- Bootstrap 方法自然保留歷史厚尾分佈，不低估極端風險

### 最大連續虧損月數
- 淨資產連續下降的最長月份
- 反映心理壓力的持續時間

### 自掏腰包機率
- 在模擬中，需要額外掏錢還款的月份比例
- 最重要的實用風險指標

## 風險展示設計

### 風險儀表板
每個標的頁面應包含：
1. 風險等級指示燈（綠/黃/紅）
2. 關鍵風險指標卡片（MDD、VaR、斷裂機率）
3. 壓力測試結果（2008 金融海嘯、2020 COVID-19 等真實歷史回測，資料來源：FinMind API 提供掛牌日起完整資料）
4. 風險比較雷達圖（與其他標的對比）

### 風險警語
根據計算結果，在結果頁面顯示動態風險警語：
- 綠色：「歷史回測顯示此策略在大多數期間可行」
- 黃色：「此策略存在一定風險，建議保留充足緊急預備金」
- 紅色：「此策略在歷史回測中有較高的虧損機率，請審慎評估」
